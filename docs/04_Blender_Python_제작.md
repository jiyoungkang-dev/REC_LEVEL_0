## 수정 기록_01

### Blender 4.4.1 컬러 매니지먼트 호환성 수정

초기 스크립트에서는 렌더링 컬러 매니지먼트 설정에서 `Filmic` View Transform을 사용하도록 작성했다.

하지만 Blender 4.4.1 환경에서 `Filmic` 항목이 존재하지 않아 다음 오류가 발생했다.

```text
TypeError: bpy_struct: item.attr = val: enum "Filmic" not found
```

이를 해결하기 위해 `set_color_management()` 함수를 추가하고, View Transform을 `AgX`, `Filmic`, `sRGB` 순서로 적용하도록 수정했다.

Blender 4.x 환경에서는 `AgX`가 기본 컬러 매니지먼트로 사용되는 경우가 많기 때문에, `AgX`를 우선 적용하도록 변경했다. 또한 Blender 버전별 설정 차이로 인해 스크립트가 중단되지 않도록 `try-except` 구조를 사용했다.

이번 수정으로 Blender 4.4.1에서도 미로형 백룸 기본 세트 생성 스크립트가 안정적으로 실행될 수 있도록 개선했다.

# 수정 기록_02

### 테스트 렌더 밝기 및 조명 방향 수정

미로형 백룸 기본 세트 생성 후 첫 테스트 렌더를 진행했으나, 화면이 지나치게 어둡게 출력되는 문제가 발생했다.

이를 해결하기 위해 형광등 조명 에너지 값을 전체적으로 상향하고, 시작 지점 근처에 약한 보조 Area Light를 추가했다. 또한 Blender 4.4.1 환경에서 `Filmic` View Transform이 존재하지 않는 문제를 고려해 `AgX`를 우선 적용하도록 컬러 매니지먼트 설정을 수정했다.

이를 해결하기 위해 다음 항목을 수정했다.

- Area Light의 X축 180도 회전을 제거
- 형광등 조명 에너지 값을 전체적으로 상향
- 카메라 시작 위치와 바라보는 방향 조정
- 렌더 노출값을 `-0.15`에서 `0.25`로 수정
- 월드 배경색을 약간 밝게 조정

이번 수정으로 테스트 렌더에서 벽, 바닥, 천장, 복도 구조를 확인할 수 있도록 개선했다.
