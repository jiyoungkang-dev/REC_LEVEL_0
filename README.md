# REC: LEVEL 0

**REC: LEVEL 0**는 Blender를 활용해 제작하는 1인칭 백룸 호러 단편영화 프로젝트입니다.

본 프로젝트는 백룸 특유의 반복적인 노란 공간, 낮은 천장, 형광등, 축축한 카펫, 불안정한 카메라 움직임을 중심으로 심리적 공포 분위기를 구현하는 것을 목표로 합니다.

단순히 하나의 3D 장면을 제작하는 것이 아니라, 기획부터 공간 설계, Blender Python을 활용한 씬 자동 생성, 절차적 쉐이딩, 조명 연출, 카메라 워킹, 렌더링까지 전체 제작 과정을 하나의 포트폴리오 프로젝트로 정리합니다.

## 기획 의도

이 프로젝트는 익숙하지만 비정상적인 공간이 주는 불안감을 시각적으로 표현하는 것을 목표로 합니다.

백룸 특유의 노란 벽지, 축축한 카펫, 낮은 천장, 반복되는 형광등을 3D 공간 안에서 구현하고, 카메라가 직접 그 공간을 걷는 듯한 1인칭 시점으로 연출합니다.

공포의 핵심은 직접적인 괴물 표현보다 공간 자체가 주는 압박감에 있습니다. 반복되는 구조, 미묘하게 달라지는 통로, 깜빡이는 형광등, 보이지 않는 존재감 등을 통해 관객이 길을 잃은 듯한 감각을 느끼도록 구성합니다.

## 제작 목표

본 프로젝트의 제작 목표는 다음과 같습니다.

1. Blender에서 백룸 기반의 모듈형 실내 공간을 제작합니다.
2. Blender Python을 사용해 바닥, 벽, 천장, 기둥, 형광등을 자동 배치합니다.
3. 절차적 머티리얼을 활용해 벽지, 카펫, 천장 타일, 형광등 재질을 제작합니다.
4. 형광등 깜빡임과 어두운 복도 연출을 통해 공포 분위기를 강화합니다.
5. 1인칭 핸드헬드 카메라 워킹을 구현합니다.
6. 최종적으로 짧은 호러 단편영상을 렌더링합니다.
7. 제작 과정을 문서화하여 포트폴리오 자료로 정리합니다.

## 주요 구현 요소

### 1. 모듈형 백룸 환경

백룸 공간을 하나의 거대한 맵으로 제작하지 않고, 방, 복도, 코너, 막다른 공간을 모듈 단위로 구성합니다.

이를 통해 반복되는 구조를 효율적으로 만들고, 카메라 샷에 맞춰 필요한 구간만 배치할 수 있도록 설계합니다.

### 2. Blender Python 자동화

Blender Python을 활용해 바닥, 벽, 천장, 기둥, 형광등을 자동으로 생성합니다.

반복적인 모델링 작업을 코드로 자동화하여 제작 시간을 줄이고, 구조를 쉽게 수정할 수 있도록 합니다.

### 3. 절차적 쉐이딩

이미지 텍스처에 의존하기보다 Blender의 노이즈, 컬러램프, 범프 노드 등을 활용해 벽지, 카펫, 천장 타일 재질을 제작합니다.

이를 통해 오래되고 축축한 백룸 특유의 재질감을 표현합니다.

### 4. 조명 연출

형광등을 주요 광원으로 사용하고, 일부 조명에 깜빡임 애니메이션을 적용합니다.

복도 끝이 어둡게 보이도록 조도 차이를 만들고, 공포 분위기를 강화합니다.

### 5. 1인칭 카메라 워킹

카메라는 캠코더를 든 인물의 시점처럼 움직입니다.

걷는 리듬에 맞춘 상하 흔들림, 불안한 좌우 회전, 갑작스러운 뒤돌아보기 등을 통해 아날로그 호러 느낌을 만듭니다.
